# Algorithms-part1-coursera-
Submissions of the course Algorithms Part-1.

## Structure
- `src/week3`: Collinear Points
- `src/week4`: 8 Puzzle
- `src/week5`: Kd-Trees

## Compilation and Running
Dependencies: `algs4.jar` (included in root).

To compile:
```bash
javac -cp algs4.jar -d out src/week3/*.java src/week4/*.java src/week5/*.java
```

To run 8 Puzzle Solver:
```bash
java -cp out:algs4.jar week4.Solver src/week4/puzzle04.txt
```
